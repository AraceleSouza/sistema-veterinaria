# 🐾 Cadastro de Animais - Veterinária

## 💻 Sobre o Projeto

Este projeto é um sistema simples de cadastro de animais desenvolvido em Python, com execução via terminal.
A aplicação permite registrar informações essenciais sobre animais atendidos em clínicas veterinárias ou abrigos, garantindo organização e padronização dos dados.

---

## 🚀 Funcionalidades

* 📋 Cadastro de múltiplos animais
* ✅ Validação de entradas (aceita apenas opções válidas)
* 📅 Registro automático da data do cadastro
* 📄 Geração de relatório final com os animais cadastrados

---

## 🛠️ Tecnologias Utilizadas

* Python 3
* Módulo padrão `datetime`

---

## ▶️ Como Executar o Projeto

1. Execute o arquivo principal:

```bash id="gq0l3y"
python main.py
```

2. Siga as instruções no terminal para cadastrar os animais.

---

## 📦 Campos do Cadastro

* **Nome:** nome do animal
* **Tipo:** Peixes, Mamíferos, Anfíbios, Aves ou Répteis
* **Sexo:** Masculino (M) ou Feminino (F)
* **Vacinado:** Sim (S) ou Não (N)
* **Descrição:** mínimo de 10 caracteres
